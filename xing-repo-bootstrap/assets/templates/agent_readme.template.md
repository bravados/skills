# Agentic Coding Guide

This folder contains short, task-focused docs used by Codex.

## Documents
- `PLANS.md` - execution plan template
- `SETUP.md` - local setup instructions
- `TESTING.md` - test commands and expectations
- `CODING_STYLE.md` - style rules not enforced by formatters
- `ARCHITECTURE.md` - module/service boundaries and data flow
- `GLOSSARY.md` - company acronyms and internal terms
- `standards/` - scoped standards used by this repo

## How to use
- Start with `AGENTS.md` at repo root.
- Open relevant `.agent/` docs based on the task.
- Load only the standards needed for the change.
- See `AGENTS.md` for the repository structure overview and key commands.
